Local Companion

This helper runs on your computer at 127.0.0.1:8765 so the web app can scan folders and apply decisions without uploading photos.

  1. Open a terminal on your computer.
  2. Go to the projectâ€™s local_app folder.
  3. Install requirements (first time): 
    pip install -r requirements.txt fastapi uvicorn
  4. Run the companion: 
    python local_companion.py
  5. Copy the Companion Token printed in the console into the UI field and click Save Token.

You can set SERVER_BASE and ADMIN_TOKEN env vars to auto-register refs and auto-sort via the server:

SERVER_BASE=http://YOUR_SERVER:8080 ADMIN_TOKEN=your_token python local_companion.py
